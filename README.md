# React 掲示板アプリケーション

TechTrainの課題として作成したReact.jsを使用した掲示板アプリケーションです。

## 概要

このプロジェクトは、React.jsを使用してシンプルな掲示板機能を実装したWebアプリケーションです。

## 機能

- 投稿の一覧表示
- 新規投稿の作成
- 投稿の編集・削除

## 技術スタック

- React.js
- vite
- TailwindCSS

## セットアップ

### 必要条件

- Node.js (v22.21.1)
- npm

### 開発サーバーの起動

```bash
npm run dev
```
