# React 掲示板アプリケーション

TechTrainの課題として作成したReact.jsを使用した掲示板アプリケーションです。

## 概要

このプロジェクトは、React.jsを使用してシンプルな掲示板機能を実装したWebアプリケーションです。

## 機能

- 投稿の一覧表示
- 新規投稿の作成
- 投稿の編集・削除

## 技術スタック

- React.js
- JavaScript/TypeScript
- CSS

## セットアップ

### 必要条件

- Node.js (v18以上推奨)
- npm または yarn

### インストール

```bash
# リポジトリをクローン
git clone https://github.com/YOUR_USERNAME/React_BB.git

# ディレクトリに移動
cd React_BB

# 依存関係をインストール
npm install
```

### 開発サーバーの起動

```bash
npm start
```

ブラウザで `http://localhost:3000` を開いてアプリケーションを確認できます。

### ビルド

```bash
npm run build
```
